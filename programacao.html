🟢 Módulo 1: O Terminal no Bolso (Termux)

O Termux não é apenas um "app", ele é um sistema Linux rodando dentro do seu Android. É aqui que a mágica acontece.

1.1 Preparação do Terreno

Primeiro, não baixe o Termux pela Play Store (ela está desatualizada). Use o F-Droid. Após instalar, abra o terminal e execute os comandos de sobrevivência:

pkg update && pkg upgrade

pkg update && pkg upgrade: Atualiza todos os pacotes.

termux-setup-storage

termux-setup-storage: Dá ao Termux permissão para acessar suas pastas de fotos e documentos.

1.2 Instalando as Ferramentas de Elite

Para programar, você precisa de um motor (linguagem) e uma caixa de ferramentas (Git). Execute:

pkg install git nodejs python

🔵 Módulo 2: O Cofre e o Portfólio (GitHub)

O GitHub é onde você guarda seu código e mostra para o mundo (e para clientes) que você sabe o que está fazendo.

2.1 Configurando sua Identidade

git config --global user.name "Seu Nome"
git config --global user.email "seuemail@exemplo.com"

2.2 O Ciclo de Trabalho

git init
git add .
git commit -m "Meu primeiro site"
git push

Dica de Ouro: Use o GitHub Pages. Com ele, você hospeda seu site de graça.
