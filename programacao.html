DevMobile Roadmap

Módulo 1

O Terminal no Bolso (Termux)

O Termux é um sistema Linux rodando dentro do seu Android. É aqui que a mágica acontece.

pkg update && pkg upgrade
termux-setup-storage
pkg install git nodejs python

*Instale sempre via F-Droid para manter atualizado.

Módulo 2

O Cofre (GitHub)

Configure sua identidade para salvar seus projetos na nuvem:

git config --global user.name "Seu Nome"
git config --global user.email "email@exemplo.com"

Dica de Ouro: Use o GitHub Pages para hospedar seus sites de graça com o domínio usuario.github.io.

Módulo 3

HTML / CSS / JS

Mesmo no celular, foque no Front-end:

  • HTML5: A estrutura (ossos).
  • CSS3: O estilo (pele/roupa).
  • JavaScript: A inteligência (músculos).

Recomendo o app Acode para editar seu código com conforto no Android.

Módulo 4

Plano de Faturamento

Como transformar código em dinheiro:

1. Landing Pages: Venda para negócios locais (R$ 200 a R$ 500).

2. Freelancer: Micro-tarefas em sites como Workana e Fiverr.

3. Automação: Crie bots em Python para otimizar processos.
